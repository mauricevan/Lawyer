# ADR-0004: Continuous planning cycle and plan transitions

**Status:** Accepted  
**Date:** 2026-07-02  
**Context:** Plan10 standardizes endless improvement without plan drift.

## Decision

1. **Alignment** doc links annual themes → quarterly roadmap.
2. **Transitions** follow `plan-transition-playbook.md` + `run-plan-transition-check.sh`.
3. **KPI catalog** splits leading/lagging with corrective actions.
4. **Plan11** formal start via `plan11-kickoff.md` and `next-cycle-themes.yaml`.

## Consequences

- Quarterly retro required before roadmap lock (template provided).
- Legacy register capped at 10 open items.
- plan11 may start without waiting for all plan9 KPI measurements.
